# fitness-web

Frontend package for the fitness app.

Use the root docs as the source of truth:

- repo overview and dev workflow: [../README.md](../README.md)
- deployment notes: [../DEPLOY.md](../DEPLOY.md)

Frontend-specific files:

- `src/services/api.ts` for API calls
- `src/types.ts` for shared frontend API shapes
- `src/stores/` for app state
- `src/components/` for UI

Useful commands from the repo root:

```bash
npm run dev:web
npm run build:web
```
